### 网盘

Mac 这边网盘都是同步盘形式，没有像win下的客户端。同步盘就是桌面顶上的一个小图标，作用是同步一个文件夹，本地在那个文件夹下的文件传上去，保存到网盘的同步文件夹下的文件拉下来。然而并不好用，经常无法同步，更新不及时，你主动去同步他也很长时间找不到、不响应。

推荐度：2

推荐度：2

## 键盘流

- Alfred 2.app 
推荐度：5

键盘工作流。以前Mac自带的Spotlight并不好用，这个键盘流工具确实给大家来了很多方便，但是现在Spotlight界面做的已经很好了，也增加了一些功能，在文件（夹）的搜索和索引上要比Spotlight好用，当然Alfred可以自定义键盘流这还是Spotlight不能替代的。本人两个软件都在用，用键盘流用alt+空格用alfted，搜素文件用command+空格用Spotlight。

- Automator.app 
语言小众（Applescript，好看难写的情怀语言，当然不用这个也行）的可视化脚本工具，快速自定App，编辑工作流，复杂流程一键解决。自己编脚本的前提是熟悉本软件、熟悉AppleScript、熟悉shell编程，可能你省出来的时间远远不及你学习和调试脚本的时间，但是学习使用这个东西是绝对物有所值。
